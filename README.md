# PokeDex
Esta proyecto consiste en consumir el api https://pokeapi.co/
donde se devuelve JSON con nombre e imagen del Pokémon 

Comienza con un Pokémon aleatorio posteriormente cada 30 segundos cambia 
a un nuevo Pokémon nuevamente aleatorio, cuenta con un botón
el cual tiene como texto "Mostrar Pokémon" si evento es traer un nuevo 
Pokémon sin importa que hayan pasado los 30 segundos. Al presionar el botón 
se reinicia el contador y si no se vuelve a presionar el botón en 30 segundos 
se muestra un Pokémon aleatorio
